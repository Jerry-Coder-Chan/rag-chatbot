# RAG Documents

Place your PDF documents in this folder.

These documents will be:
1. Loaded and processed when you click "Initialize RAG System"
2. Split into chunks for efficient retrieval
3. Embedded and stored in a FAISS vector database
4. Used to answer user questions

## Supported Formats

- PDF files (.pdf)

## Note

For Streamlit Cloud deployment, you may need to include sample documents
or implement a file upload feature in the app.
